>
> 作品網址：https://the-f2e-note-app.web.app/

### 製作功能
* 新增、編輯、刪除筆記
* 加入我的最愛、垃圾桶等列表
* 可依名字、新增時間、最後修改時間進行排序
* 分享筆記

### 使用技術
* Parcel
* React
* Element-ui
* Sass
* Firebase
* Quill (Component)

### Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:1234
$ npm run dev
```
